# 🏨 Hostel and Hotel Booking System

A **Java console-based application** developed using **Object-Oriented Programming (OOP)** concepts to simplify hostel and hotel booking. The application provides secure user authentication, OTP verification, accommodation selection, payment processing, booking confirmation, receipt generation, and customer feedback through an interactive menu-driven interface.

---

## 📌 Project Overview

The **Hostel and Hotel Booking System** is designed to help users easily find and book suitable hostel or hotel accommodations. It allows users to register, log in securely, browse available accommodations based on location and category, calculate booking costs with discounts, complete payments, and receive booking confirmations.

The system also includes OTP verification for account creation and payment confirmation, making the booking process more secure and user-friendly.
